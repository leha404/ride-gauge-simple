# Welcome to your Lovable project

Личное приложение MotoApp для учета пройденного расстояния и остатка топлива

- Карта проекта и процесс разработки: `docs/project-map.md`

- npm install

- npx cap add ios
- npx cap update ios

Для корректного обновления в ios версии:
- npm run build
- npx cap sync
- npx cap run ios

Для тестирования в вебе:
npm run dev
